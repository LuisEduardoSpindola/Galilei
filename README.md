<h2>Resumo</h2>
<p>
Este estudo aborda o desenvolvimento de um software para análise do mercado financeiro em tempo real, utilizando tecnologias como C# e .NET. O objetivo é criar uma ferramenta que permita aos usuários acompanharem o mercado e auxiliar na tomada de decisões.
</p>
<p>
O projeto foca na integração com APIs financeiras para captação de dados em tempo real, como cotações de ações, índices e moedas. Foram implementados algoritmos para processamento e visualização com indicadores técnicos.
</p>
<p>
A pesquisa explora a arquitetura do software, bibliotecas utilizadas e desafios técnicos relacionados ao processamento de grandes volumes de dados.
</p>

<p><strong>Palavras-chave:</strong> Mercado Financeiro; Análise de Dados; Desenvolvimento de Software; APIs Financeiras; .NET.</p>

<hr>

<h2>Introdução</h2>
<p>
Nos últimos anos, o mercado acionário brasileiro ganhou grande destaque, tornando-se uma alternativa relevante de investimento. O número de investidores cresceu significativamente, impulsionado por fatores como a redução das taxas de juros.
</p>
<p>
Entretanto, o pequeno investidor enfrenta desafios como excesso de informação e decisões emocionais. Em contrapartida, grandes instituições utilizam automação e análise algorítmica em tempo real.
</p>
<p>
Diante disso, este trabalho propõe o desenvolvimento de um software que automatiza a coleta e análise de dados financeiros, auxiliando investidores na tomada de decisão.
</p>

<hr>

<h2>Referencial Teórico</h2>

<h3>1. Sistemas de Apoio à Decisão (SAD)</h3>
<p>
Os SADs são sistemas que combinam dados, modelos analíticos e interfaces para auxiliar decisões complexas. No contexto financeiro, permitem análise estratégica baseada em dados.
</p>

<h3>1.1 Componentes</h3>
<ul>
<li><strong>Base de Dados:</strong> Armazenamento de informações financeiras</li>
<li><strong>Base de Modelos:</strong> Algoritmos e análises técnicas</li>
<li><strong>Interface:</strong> Visualização e interação com o usuário</li>
</ul>

<h3>2. Software Financeiro Moderno</h3>
<p>
Soluções modernas utilizam APIs para integração em tempo real, permitindo automação e redução de erros manuais.
</p>

<h3>2.1 Automação</h3>
<p>
Inclui cálculo de impostos, relatórios e análise de investimentos, tornando o software mais inteligente e ativo.
</p>

<h3>3. Justificativa</h3>
<ul>
<li>Uso de SAD para suporte à decisão</li>
<li>Integração com APIs em tempo real</li>
<li>Redução da assimetria de informação no mercado</li>
</ul>

<h2>Resultados</h2>
<p>
O projeto resultou em um sistema funcional de monitoramento financeiro com dashboard intuitivo.
</p>

<ul>
<li><strong>Indicadores Econômicos:</strong> IPCA e outros dados</li>
<li><strong>Ações da B3:</strong> PETR4, VALE3, ITUB4</li>
<li><strong>Criptomoedas:</strong> Bitcoin e Ethereum</li>
</ul>

<p>
O sistema apresentou:
</p>

<ul>
<li>Baixa latência</li>
<li>Alta concorrência</li>
<li>Precisão de dados validada</li>
</ul>
